## Blank Webpack, ES6, Bootstrap 4
Get off to a quick start with this ES6 Webpack template.

![img](https://pbs.twimg.com/media/CtBI2BXVUAAXF07.jpg:large "Blank Webpack, ES6, Bootstrap 4")


### Main Components:
 - Webpack
 - Webpack Dev Server
 - Babel
 - Bootstrap 4
 - SASS


### Build it

```bash
webpack
```

### Developing it:

```bash
npm run dev

```
